我常用的软件及小工具

2020-08-23

我常用的软件及小工具，能够提高一定程度的工作效率

我常用的软件及工具

Desktop

  1. Everythin - Windows平台下最好的文件搜索工具，用于搜索冗余文件非常方便
  2. Bandizip v6.25 - 这款压缩软件无广告效率高，基本装机必备！可惜的是7.0开始就收费制了，而且还挺贵，因此6.25是最好的免费版压缩软件
  3. Quicker - 这款软件主要用于我常用的软件启动，因为我的桌面是除了此电脑外不保留任何图标快捷方式的。通过这是Alt+Space我就能从这个软件启动我需要的软件。现在这款软件也已经开启订阅制收费，但好在收费不高，尚能接受。截至2020年8月23日，他的年费尚且是57.6元
  4. Notion - 这是一款跨平台的笔记软件，有非常高效，牛逼的无限冗余记录，好像被某个大公司收购，个人用户基本功能使用完全没问题
  5. Laragon - 这个软件我也是刚接触，他对于开发环境的集成非常高，基本目录结构也是linux式的，虽然主要目标是开发laravel，但其他开发能力也还不错
  6. BatteryBar - 笔记本电池监测工具，有比较贴切的提醒和数据显示，这款软件已经多年未更新，但依然好用。我在52pojie上发布过他的3.6.6版，有兴趣的朋友可以去看看
  7. HWINFO - 比AIDA64更直接的电脑实时数据监视工具，对于电脑发烧友来说必不可少
  8. atrofac - 这是用于华硕ROG幻14系列笔记本的风扇控制软件，在Github上开源的。由于华硕本身的ARMOURY CRATE对风扇的控制就缺少，用这个软件的主要目的是在笔记本低负载下完全停止风扇
  9. Honeyview - 同样出自Bandizip软件团队的另一款看图神器，因为win10自带的看图应用真的垃圾，随便打开个4k图就要半天，在这款软件上基本秒开，而且有非常强大的预览功能，例如PS文件可直接打开查看（不能编辑），方便你查找文件
  10. Windows Termianl - 这个不用多说，微软少数的优秀软件之一，我还写过他的一些应用，虽然现在最新版的WT已经在右键菜单加入它了，但还是不够方便
  11. Netch - 游戏代理用的，当然是用来玩国际服游戏，不过首先你需要有一个速度优异的境外服务器，也是Github开源项目
  12. ACG播放器 - 微软商店里最好的播放器，虽然免费有广告，但当年刚收费时也不贵，就17元
  13. WinSCP - SFTP/FTP等文件传输软件
  14. Postman - HTTP API测试开发
  15. Navicat - 数据库管理软件
  16. Snipaste - 截图与贴图，好用且快捷，免费版就够
  17. electerm - SSH终端，xshell的最佳替代品，同样Github开源
  18. Ditto - 非常好用的剪切板记录工具，默认能保存最近的500条记录，微软商店有下
  19. Bitwarden - 绝无仅有的开源免费密码管理软件，你可以自己搭建服务器，也可以用他们提供的服务
  20. IINA - 这是我在Mac上唯一选择的视频播放软件，至今找不到他的替代品，简单来说就是无敌，同样是Github开源作品。
  21. NTFS Tools - 同样是免费开源的软件，能在MacOS上使用NTFS的硬盘

Chrome插件

  1. Adblock Plus - 免费的广告拦截器
  2. Bitwarden - 免费密码管理器
  3. FeHelper - 前端助手，JSON自动格式化、手动格式化，支持排序、解码、下载等
  4. Tampermonkey - 脚本神器，不必多说
  5. Imagus - 鼠标移到图片链接自动加载放大，很实用
  6. v2ex plus - v2站的扩展

其实不难发现很多开源软件都是非常优秀的，完全不亚于所谓大公司的大制作。

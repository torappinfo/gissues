在Ubuntu/Debian中安装Transmission

2020-03-15

Transmission是一种BitTorrent客户端,老司机们必不可少的得力工具之一，也是PT友做种的首选。这里简单介绍Debian\Ubantu下安装Transmission的教程。

安装

更新系统包：

sudo apt-get update

安装Transmission：

apt-get install transmission-daemon -y

配置：

停止服务/etc/init.d/transmission-daemon stop

编辑配置文件nano /var/lib/transmission-daemon/info/settings.json

需要修改的如下：

"download-dir"			# 下载目录
"rpc-password"			# 访问密码
"rpc-username"			# 登录用户名
"rpc-whitelist": "*"	# 通过ip访问则需要允许所有ip访问
"rpc-port"				# 访问端口
...

建议修改端口并用nginx + 域名反向代理，这样不用修改上述白名单，保持127.0.0.1即可

确认保存后启动/etc/init.d/transmission-daemon start

服务式命令service transmission-daemon start/stop

卸载sudo apt-get autoremove --purge transmission-daemon

美化

获取美化脚本兵运行wget https://raw.githubusercontent.com/ronggang/transmission-web-control/master/release/install-tr-control.sh

sh install-tr-control.sh

安装默认按1

重新进入就可以看到新的界面及选项

Debian的安装只需要不加命令前的sudo即可。
